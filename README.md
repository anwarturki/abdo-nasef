# تطبيق عبدو ناصف

موقع ويب شخصي/رسمي لعبدو ناصف

## المميزات
- واجهة بسيطة وسهلة الاستخدام
- متوافق مع جميع الأجهزة
- يعمل كتطبيق PWA (يمكن تثبيته على الهاتف)

## كيفية الاستخدام
1. افتح [الموقع](https://anwarturki.github.io/abdo-nasef/)
2. اضغط على "تثبيت" في متصفح Chrome

## التقنيات المستخدمة
- HTML5
- CSS3
- JavaScript
